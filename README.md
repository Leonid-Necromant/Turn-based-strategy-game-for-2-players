Тактическая пошаговая игра, вдохновлённая классическими стратегиями. Игроки управляют отрядами из четырёх типов юнитов на карте с процедурно генерируемым ландшафтом.

Геймплей

- Два игрока: Красный и Синий
- Ходят по очереди
- Побеждает тот, кто уничтожит все юниты противника

Действия за ход
За один ход можно:
1. Переместить один юнит (на его максимальную дальность)
2. Атаковать врага, если он в радиусе действия
3. Закончить ход

После перемещения атаковать тем же юнитом нельзя — нужно выбирать стратегию!

Юниты

Лучник
Катапульта
Всадник
Мечник

Карта

- Размер: 14×14 клеток
- Типы клеток:
  - Трава — проходима
  - Вода — непроходима
- Ландшафт генерируется процедурно при запуске:
  - Добавляются 2 случайные реки
  - В реках есть «разрывы» для прохода

Визуальное оформление

- Все изображения генерируются автоматически при первом запуске
- Папка draw/ создаётся автоматически, в неё сохраняются текстуры:
  - Трава и вода
  - Иконки юнитов
- Можно заменить стандартные изображения, положив свои .png файлы в draw/:
  - grass.png, water.png
  - archer.png, catapult.png, horseman.png, swordsman.png

Установка и запуск

Требования
- Python 3.8+
- Pygame

bash

git clone https://github.com/Leonid-Necromant/turn-based-strategy-game.git
cd turn-based-strategy-game

Установите зависимости
pip install -r requirements.txt
